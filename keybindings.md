# Keybindings

# Terminal

| Горячие клавиши     | Действие                         |
| ------------------- | -------------------------------- |
| F2                  | Переименовать файл               |
| F11                 | Развернуть на весь экран         |
| win + <-            | Разместить окно терминала слева  |
| win + ->            | Разместить окно терминала справа |
| Ctrl + Alt + T      | открыть терминал                 |
| Ctrl + Alt + Delete | меню выключения                  |
| Ctrl + W (Alt + F4) | закрыть окно                     |
